1. A cosa serve bootstrap?
Bootstrap è un framework front-end gratuito e open source che permette di creare siti web responsive in modo rapido e semplice. Fornisce una griglia CSS, componenti pronti all’uso (bottoni, form, navbar, ecc.) e utility per lo stile.

3. Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?
Un CDN (Content Delivery Network) è una rete di server distribuiti geograficamente che servono file statici (CSS, JS, immagini). I vantaggi sono: velocità (file caricati dal server più vicino), affidabilità, riduzione carico server e cache globale.
4. A cosa serve questo attributo del tag meta?
content="width=device-width, initial-scale=1"
Serve a rendere la pagina responsive sui dispositivi mobili: imposta la larghezza della viewport uguale alla larghezza dello schermo e lo zoom iniziale al 100%.
5. Che differenza c’è tra Container e Container Fluid?
Container: ha larghezza massima fissa (che cambia con lo schermo) e margini laterali. Ideale per contenuti centrati.
Container Fluid: occupa il 100% della larghezza dello schermo senza margini. Ideale per layout full-width.
Container (limitato)

Occupa larghezza massima fissa.

Container Fluid (full width)
